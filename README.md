````markdown
## Tools Used

- React  
- Redux  
- React Router v6  
- React Hooks  
- Country API  
- SASS (for styling)  

---

## Features

- Fetch data from [Rest Country API](https://restcountries.com)  
- Multi-page routing with React Router  
- Search countries by name  
- Filter countries by region  
- Dark/Light theme toggle  
- Load more button for countries  
- Fully responsive UI

   ```bash
````
